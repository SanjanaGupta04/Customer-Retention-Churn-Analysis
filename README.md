# Customer Retention & Churn Analysis

## Project Overview
This project analyzes customer churn behavior using the Telco Customer Churn Dataset. The objective is to identify churn patterns, retention drivers, and business strategies to improve customer retention.

## Tools Used
- Power BI
- Excel

## Dashboard Features
- Total Customers KPI
- Churned Customers KPI
- Active Customers KPI
- Churn Rate KPI
- Customer Churn by Contract Type
- Customer Churn by Payment Method
- Customer Churn by Tenure Group
- Internet Service Analysis
- Interactive Slicers

## Key Insights
- Customers with Month-to-Month contracts have the highest churn.
- Most customer churn occurs within the first 12 months.
- Electronic Check users show higher churn rates.
- Fiber Optic customers represent a large share of churned customers.

## Recommendations
- Encourage long-term contracts.
- Improve onboarding and customer engagement.
- Launch targeted retention campaigns.
- Develop loyalty programs for long-term customers.
